API Access Guide

Use your personal API key directly with the PathomAI API

Every API key is generated per account. Use it in the X-API-Key header. You do not need to send email, username, or password in your scripts.

Base URL:
Authentication

Send X-API-Key: pthm_... on every request.

Tenant Scope

The key is tied to your account and tenant, so jobs created by API also appear in the UI.

Security

Treat the key like a password. Store it in environment variables, CI secrets, or a password manager.

Quick Start

Create an API key from Settings, copy it once, and store it as an environment variable.

Windows PowerShell

$env:PATHOMAI_API_KEY="pthm_your_real_key_here"
$env:PATHOMAI_BASE_URL=""

Linux or macOS

export PATHOMAI_API_KEY="pthm_your_real_key_here"
export PATHOMAI_BASE_URL=""
All examples below assume you already set PATHOMAI_API_KEY. Replace sample job IDs and file paths with your real values.

Core Endpoints

GET /api/videos
List jobs visible to your tenant.
GET /api/videos/{job_id}
Get one job with transcript, summary, sentiment, and action items.
POST /api/videos/upload
Upload a local video file with multipart form data.
POST /api/videos/import
Import a video from a public URL.
POST /api/videos/{job_id}/retry
Retry a failed processing job.
POST /api/videos/{job_id}/summary/regenerate
Generate a focused summary for one part of a transcript.
POST /api/videos/{job_id}/chat
Ask a transcript-grounded question about a video.
POST /api/videos/{job_id}/chat/suggestions
Get follow-up questions for the same job.

Windows PowerShell Examples

List Jobs

$headers = @{ "X-API-Key" = $env:PATHOMAI_API_KEY }
Invoke-RestMethod `
  -Method Get `
  -Uri "$env:PATHOMAI_BASE_URL/api/videos" `
  -Headers $headers

Upload a Local Video

$headers = @{ "X-API-Key" = $env:PATHOMAI_API_KEY }
$form = @{
  file = Get-Item "C:\videos\meeting.mp4"
  language_hint = "auto"
}
Invoke-RestMethod `
  -Method Post `
  -Uri "$env:PATHOMAI_BASE_URL/api/videos/upload" `
  -Headers $headers `
  -Form $form

Import a Video by URL

$headers = @{
  "X-API-Key" = $env:PATHOMAI_API_KEY
  "Content-Type" = "application/json"
}
$body = @{
  video_url = "https://example.com/video.mp4"
  language_hint = "auto"
} | ConvertTo-Json

Invoke-RestMethod `
  -Method Post `
  -Uri "$env:PATHOMAI_BASE_URL/api/videos/import" `
  -Headers $headers `
  -Body $body

Ask AI Chat

$jobId = "replace-with-job-id"
$headers = @{
  "X-API-Key" = $env:PATHOMAI_API_KEY
  "Content-Type" = "application/json"
}
$body = @{
  question = "What decisions were made?"
  chat_history = @()
  asked_questions = @()
} | ConvertTo-Json -Depth 5

Invoke-RestMethod `
  -Method Post `
  -Uri "$env:PATHOMAI_BASE_URL/api/videos/$jobId/chat" `
  -Headers $headers `
  -Body $body

Linux and macOS curl Examples

List Jobs

curl "$PATHOMAI_BASE_URL/api/videos" \
  -H "X-API-Key: $PATHOMAI_API_KEY"

Upload a Local Video

curl "$PATHOMAI_BASE_URL/api/videos/upload" \
  -H "X-API-Key: $PATHOMAI_API_KEY" \
  -F "file=@./meeting.mp4" \
  -F "language_hint=auto"

Import a Video by URL

curl "$PATHOMAI_BASE_URL/api/videos/import" \
  -H "X-API-Key: $PATHOMAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "video_url": "https://example.com/video.mp4",
    "language_hint": "auto"
  }'

Focused Summary

JOB_ID="replace-with-job-id"

curl "$PATHOMAI_BASE_URL/api/videos/$JOB_ID/summary/regenerate" \
  -H "X-API-Key: $PATHOMAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "instruction": "Summarize only the student issue discussion and list the action items."
  }'

Get Follow-up Suggestions

curl "$PATHOMAI_BASE_URL/api/videos/$JOB_ID/chat/suggestions" \
  -H "X-API-Key: $PATHOMAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "asked_questions": ["What decisions were made?"]
  }'

Python requests Example

import os
import requests

BASE_URL = os.environ["PATHOMAI_BASE_URL"]
API_KEY = os.environ["PATHOMAI_API_KEY"]
HEADERS = {"X-API-Key": API_KEY}

# List jobs
jobs = requests.get(f"{BASE_URL}/api/videos", headers=HEADERS, timeout=60)
jobs.raise_for_status()
print(jobs.json())

# Import by URL
payload = {
    "video_url": "https://example.com/video.mp4",
    "language_hint": "auto",
}
response = requests.post(
    f"{BASE_URL}/api/videos/import",
    headers={**HEADERS, "Content-Type": "application/json"},
    json=payload,
    timeout=60,
)
response.raise_for_status()
print(response.json())

Node.js fetch Example

const BASE_URL = process.env.PATHOMAI_BASE_URL;
const API_KEY = process.env.PATHOMAI_API_KEY;

const headers = {
  "X-API-Key": API_KEY,
  "Content-Type": "application/json",
};

const response = await fetch(`${BASE_URL}/api/videos/import`, {
  method: "POST",
  headers,
  body: JSON.stringify({
    video_url: "https://example.com/video.mp4",
    language_hint: "auto",
  }),
});

if (!response.ok) {
  throw new Error(await response.text());
}

console.log(await response.json());

Errors and Validation

  • 401 Authentication required: no bearer session or X-API-Key was sent.
  • 401 Invalid API key: the key is wrong, revoked, or no longer valid.
  • 400 Unsupported language hint: allowed values are auto, en, english, tl, tagalog.
  • 400 Unsupported video file type: only supported video extensions can be uploaded.
  • 404 Video job not found: the job is not in your tenant scope.
  • 409 Only failed jobs can be retried: retry works only on failed jobs.
  • Focused summary validation: instruction must be between 5 and 2000 characters.
